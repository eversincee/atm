# Симулятор банкомата

Консольное приложение на Python, моделирующее работу банкомата.

## Запуск

```bash
cd atm
python main.py
```

## Данные по умолчанию

| | |
|---|---|
| Номер карты | `1234567890123456` |
| PIN | `1234` |
| Баланс | 10 000 ₽ |
| Пароль администратора | `admin` |

## Возможности

**Пользователь:** проверка баланса, снятие/внесение наличных, перевод на другую карту, история операций.

**Администратор:** создание счётов и карт, пополнение кассы, просмотр всех счётов, разблокировка карт.

## Структура

```
atm/
├── main.py         # точка входа и консольный UI
├── models/         # Transaction, Account, Card
├── exceptions.py   # иерархия исключений
├── storage.py      # SQLite (data/atm.db)
├── bank.py         # управление счетами и картами
└── atm.py          # операции банкомата
```
